## Why Use This Project?

Building and maintaining a large collection of card images can be manual and time-consuming. `ygo-card-image-hosting` automates this process, saving you significant effort.

## Key Features at a Glance

<CardGrid stagger>
  <Card title="Terraform Automation" icon="seti:terraform">
    Automates the setup of AWS S3 and CloudFront infrastructure.
  </Card>
  <Card title="Smart Archiving" icon="seti:image">
    Skips existing images in S3 on subsequent runs for efficiency.
  </Card>
  <Card title="Date & Size Filtering" icon="setting">
    Configure downloads based on card release dates and image sizes.
  </Card>
  <Card title="CDN Delivery" icon="rocket">
    Serve images globally via AWS CloudFront for fast access.
  </Card>
  <Card title="Dockerized" icon="seti:docker">
    Runs in a consistent environment using Docker and Docker Compose.
  </Card>
</CardGrid>
